Política de Privacidade

📅 Vigência: Abril de 2026
📱 App: Match Time Pro

Esta Política de Privacidade descreve como o aplicativo Match Time Pro coleta, utiliza e protege as informações dos seus usuários. Ao usar o aplicativo, você concorda com os termos descritos neste documento.

Levamos a privacidade muito a sério. O Match Time Pro foi projetado para funcionar com o mínimo de dados pessoais possível, sendo a maioria das funcionalidades disponível sem qualquer cadastro.

3. Como Utilizamos os Dados

  • Autenticação: dados de conta usados exclusivamente para identificar o usuário logado.
  • Notificações push: token FCM usado para alertas de início de partida, gols, cartões e fim de jogo.
  • Publicidade: AdMob usa o identificador de publicidade para exibir anúncios relevantes. Pode ser desativado nas configurações do Android.
  • Analytics: dados agregados e anonimizados para identificar falhas e melhorar o app.
  • Dados locais: times favoritos, histórico de pesquisa e preferências permanecem no dispositivo para personalizar a experiência.

⚠️ O Match Time Pro não vende, aluga nem compartilha dados pessoais com terceiros para fins comerciais próprios.

4. Serviços de Terceiros

O aplicativo integra serviços de terceiros com políticas de privacidade próprias.

Google Firebase (Authentication, Cloud Messaging, Analytics)

Política: firebase.google.com/support/privacy

Google AdMob

Pode coletar o identificador de publicidade para exibir anúncios personalizados.
Política: policies.google.com/privacy
Para desativar: Configurações Android → Google → Anúncios → Desativar personalização.

API-Football (API Sports)

Fornece dados de partidas. As requisições passam pelo nosso servidor intermediário, que envia apenas parâmetros de consulta (IDs de time, liga, temporada). Nenhum dado pessoal do usuário é transmitido.
api-football.com

GNews API

Fornece notícias de futebol. As requisições passam pelo nosso servidor intermediário, que envia apenas termos de busca. Nenhum dado pessoal do usuário é transmitido.
gnews.io/privacy-policy

Servidor intermediário (backend)

O Match Time Pro utiliza um servidor próprio que faz cache das informações esportivas, evitando chamadas redundantes às APIs externas. O servidor armazena apenas o token FCM (para envio de notificações) e dados públicos de partidas. Não armazena dados pessoais identificáveis.

5. Retenção e Exclusão de Dados

  • Conta Firebase: mantida enquanto ativa. O usuário pode excluir a conta em Configurações → Conta → Excluir conta.
  • Token FCM: renovado periodicamente pelo sistema, sem vínculo com dados identificáveis. Tokens inválidos são removidos automaticamente do servidor.
  • Dados locais: apagados ao desinstalar o aplicativo.
  • AdMob e Analytics: gerenciados pelo Google conforme suas políticas de retenção.

6. Direitos do Usuário (LGPD)

Em conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018), você tem direito a:

  • Acesso: saber quais dados temos sobre você.
  • Correção: corrigir dados incompletos ou incorretos.
  • Exclusão: solicitar a exclusão dos seus dados pessoais.
  • Revogação de consentimento: retirar o consentimento a qualquer momento.
  • Portabilidade: solicitar seus dados em formato legível por máquina.
  • Informação sobre compartilhamento: saber com quais entidades seus dados são compartilhados.

Para exercer esses direitos, entre em contato pelo e-mail da Seção 1.

7. Uso por Crianças

O Match Time Pro não é direcionado a crianças menores de 13 anos e não coleta intencionalmente dados de menores. Caso identifiquemos tal situação, os dados serão removidos imediatamente.

8. Segurança dos Dados

  • Comunicação com APIs externas via HTTPS/TLS.
  • Autenticação gerenciada pelo Firebase Authentication.
  • Dados locais no armazenamento privado do aplicativo.
  • Nenhuma senha armazenada pelo desenvolvedor.

9. Alterações nesta Política

Esta política pode ser atualizada periodicamente. Alterações significativas serão comunicadas via notificação no app ou por e-mail. O uso continuado após alterações constitui aceitação dos novos termos.
